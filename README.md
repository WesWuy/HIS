# Human Improvement Systems (HIS)

Human Improvement Systems is the operating system for the Captain Wes AI-powered coaching brand.

The purpose of this repo is to build the backend automation stack for a growing Instagram-based revenue engine using:

- Claude Code
- Codex
- Instagram
- Gmail
- Google Sheets
- Google Drive
- Google Apps Script

## Core Strategy

HIS should not behave like a generic motivational page. It should operate like a media + lead-generation + coaching system.

The stack is designed to:

1. Generate and organize content ideas.
2. Turn approved ideas into scripts, captions, carousels, and reels.
3. Capture leads through a Human Improvement Scorecard.
4. Generate AI-assisted reports.
5. Send Gmail-based follow-up sequences.
6. Track lead quality, offers, and revenue signals.
7. Keep Captain Wes as the recognizable character/IP asset.

## Automation Principle

Do not automate spam.

Safe automation:

- content planning
- script generation
- asset organization
- lead capture
- email delivery
- follow-up tracking
- reporting

Avoid:

- fake engagement
- mass follows
- mass likes
- scraping private data
- unsolicited DM spam
