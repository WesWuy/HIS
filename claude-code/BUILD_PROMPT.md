# Claude Code Build Prompt

Use this prompt in Claude Code to build the first working version of the Human Improvement Systems backend.

```text
You are building the backend operating system for an Instagram-based AI coaching brand called Human Improvement Systems.

The character is Captain Wes: a Digital Crusader / Human Upgrade Engineer helping people build discipline, energy, clarity, and self-control using AI, psychology, biohacking, and personal development systems.

Build a Google Workspace automation system using Google Sheets, Google Drive, Gmail, and Google Apps Script.

Requirements:

1. Create a Drive folder structure:
- HIS/01_Brand_Bible
- HIS/02_Content_Ideas
- HIS/03_Scripts
- HIS/04_Carousels
- HIS/05_Reels
- HIS/06_Posting_Calendar
- HIS/07_Leads
- HIS/08_Offers
- HIS/09_Client_Deliverables
- HIS/10_Analytics

2. Create a Google Sheet named HIS Command Center with tabs:
- Content Pipeline
- Lead Tracker
- Offer Tracker
- Prompt Library
- Analytics Dashboard
- Customer Journey

3. Build Apps Script functions to:
- create the Drive folder structure
- create spreadsheet tabs and headers
- seed starter rows for offers and customer journey
- create Gmail drafts for lead follow-up
- process scorecard form responses
- append leads to Lead Tracker
- send lead magnet email

4. Do not use paid APIs yet.

5. Keep Instagram posting manual at first. The system should prepare content and track status, not spam or fake engagement.

6. Add beginner-friendly setup instructions.

7. Make the code safe, readable, and easy to modify.
```

## Expected Output

Claude Code should produce:

- finalized Apps Script code
- setup instructions
- Google Form field recommendations
- testing checklist
- deployment checklist
