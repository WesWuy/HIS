# Google Sheets Schema

Create one spreadsheet named:

`HIS Command Center`

## Tabs

1. Content Pipeline
2. Lead Tracker
3. Offer Tracker
4. Prompt Library
5. Analytics Dashboard
6. Customer Journey

## Content Pipeline Columns

| Column | Purpose |
|---|---|
| Date | Intended post date |
| Platform | Instagram, YouTube Shorts, TikTok, LinkedIn |
| Content Type | Reel, Carousel, Story, Static, Email |
| Pillar | Discipline, AI, Biohacking, Mindset, Spiritual, Brotherhood |
| Hook | First-line attention grabber |
| Script | Spoken/video script |
| Caption | Instagram caption |
| CTA | Comment, DM, link, scorecard, call booking |
| Status | Idea, Drafting, Review, Ready, Posted, Repurpose |
| Asset Link | Google Drive asset URL |
| Posted Link | Final post URL |
| Notes | Internal notes |

## Lead Tracker Columns

| Column | Purpose |
|---|---|
| Timestamp | Form submission time |
| Name | Lead name |
| Email | Lead email |
| Instagram Handle | IG handle |
| Scorecard Answers | Raw answers |
| AI Summary | AI-generated insight |
| Lead Score | 1-100 |
| Recommended Offer | Free, Low, Mid, High Ticket |
| Follow-Up Status | New, Sent, Replied, Booked, Won, Lost |
| Notes | Human notes |

## Offer Tracker Columns

| Column | Purpose |
|---|---|
| Offer Name | Product/service |
| Price | Price point |
| Promise | Main outcome |
| Audience | Who it serves |
| CTA | Next action |
| Delivery Method | PDF, email, call, course, membership |
| Status | Draft, Testing, Active, Retired |

## Prompt Library Columns

| Column | Purpose |
|---|---|
| Prompt Name | Internal label |
| Use Case | Script, caption, email, report, analysis |
| Prompt | Full reusable prompt |
| Output Format | Markdown, JSON, table, email |
| Notes | Refinements |

## Analytics Dashboard Metrics

- Posts published
- Reels published
- Carousels published
- Leads captured
- Email open/reply tracking notes
- Calls booked
- Offers made
- Sales closed
- Revenue
- Best hooks
- Best CTAs
